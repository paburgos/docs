# Equipment Template Update

Creating an Equipment Template
### Search a Equipment Type:

-Use the search functionality to find the specific type of equipment you want to create a template for.
### Select Manufacturer:
-Choose the appropriate manufacturer from the dropdown list.
### Select Status:
-Define the status of the equipment template (e.g., Approved, Pending).
### Add Device Class:
-Specify the class to categorize the device appropriately.
### Add Model:
-Enter the model number to uniquely identify the equipment.
### Add Tolerance Type:
-If required, select and add the tolerance type depending on the equipment type.
### Add Test Points:
-Define test points for calibration based on the equipment type
