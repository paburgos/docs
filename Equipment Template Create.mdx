# Equipment Template Create

o	Search a Equipment Type: : Find the type of equipment you want to create a template for.

o	Select Manufacturer: Choose the manufacturer of the equipment.

o	Select Status: Define the status of the equipment (e.g., Approved, Pending).

o	Add Device Class: Specify the device class for categorization.

o	 Add Model: Enter the model number of the equipment.

o	Add Tolerance Type: If necessary, add the tolerance type based on the equipment type.

o	Add Test Points: Define test points depending on the equipment type.
